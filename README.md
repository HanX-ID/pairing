## Instalasi
1. **Install paket**
   ```bash
   pkg update && pkg upgrade -y
   pkg install git -y
   pkg install nodejs-lts -y
   ```
2. **Clone repository**
   ```bash
   git clone https://github.com/HanX-ID/pairing
   cd pairing
   ```
3. **Jalankan script**
   ```bash
   npm install
   npm start
   ```
